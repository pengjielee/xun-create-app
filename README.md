一个Nodejs脚手架，可以快速创建你的项目。

## 全局安装安装

```bash
$ npm install xun-create-app --global
```

## 使用

```bash
$ xun-create-app

or

$ xca
```


## 当前支持的项目模板

- react
- react-ts
- react-vite
- react-ts-vite
- react-parcel
- vue2
- vue3
- vue3-vite
- electron
- express-api
- express-hbs
- express-upload，
- koa-static
- miniprogram-wechat
